## ✨ 特徴

- **📚 学習モード**: カードをめくりながら、上の句・下の句・作者・解説を学習
- **✏️ クイズモード**: 4択クイズで理解度をチェック
- **⚡ タイムアタック**: 60秒以内に何問解けるか挑戦
- **🎨 季節別グラデーション**: 春・夏・秋・冬・恋・海などテーマに応じた美しい背景
- **👤 作者メモ**: 全100首に歴史的背景や作者の逸話を収録
- **💾 学習記録**: 覚えた歌、最高スコア、正解率を自動保存
- **📱 レスポンシブ対応**: スマホ・タブレット・PCで快適に学習

## 🎯 使い方

1. [アプリを開く](https://yokko405.github.io/hyakunin-isshu/百人一首アプリ.html)
2. 3つのモードから選択
   - **学習モード**: カードをタップしてめくり、解説を読む
   - **クイズモード**: 上の句から下の句を選ぶ
   - **タイムアタック**: 制限時間内に連続正解を目指す
3. 学習の進捗は自動的に保存されます

## 🌸 季節別テーマ

アプリは各歌のテーマを自動判定し、適切な背景色を表示します：

- **春** 🌸 - 桜色のグラデーション
- **夏** 🎆 - 青空と海のグラデーション
- **秋** 🍁 - 紅葉色のグラデーション
- **冬** ❄️ - 雪と氷のグラデーション
- **恋** 💕 - ピンク紫のグラデーション
- **海** 🌊 - 青緑のグラデーション
